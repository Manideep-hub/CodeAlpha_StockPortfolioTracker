  # 📈 Stock Portfolio Tracker

A simple Python application developed as part of the **CodeAlpha Python Programming Internship**. This project allows users to calculate the total value of their stock portfolio using predefined stock prices.

## 🚀 Features

- Display available stock prices
- Accept stock names and quantities from the user
- Validate stock names
- Calculate investment for each stock
- Calculate the total investment value
- Generate a portfolio summary
- Save the portfolio report to a text file (`investment_report.txt`)

## 🛠️ Technologies Used

- Python 3
- Dictionaries
- Loops
- Conditional Statements
- User Input
- File Handling
